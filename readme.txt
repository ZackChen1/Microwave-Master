Microwave Master is a highly sophisticated program that utilizes advanced mathematical algorithm and thermodynamics in hope of providing our users the optimal time settings for your microwave delicacies.

###IMPORTANT###
1. Please note that, due to several technical difficulties, the program only allows the input format of numbers.
2. Also, with the same reasoning listed above, please refrain yourself from inputting time formats other than seconds.
3. Customization of the program is currently not supported, please take note of future updates and patches.
